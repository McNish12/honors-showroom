# Start Here: Honors Showroom Firebase Admin MVP

Nik, start here tomorrow.

## Big Picture

The current Honors Showroom is a working static catalog backed by Google Sheets. We are using this repo as an ecommerce learning project.

The goal is to learn how Honors could manage online products, stock, customer verbiage, artwork uploads, and eventually a simple Honors Plaques designer.

## Your First Goal

Your first goal is understanding the project, not coding.

Before building anything, fill out:

`FIREBASE_ADMIN_PLAN.md`

## Branch

Work on:

`firebase-admin-mvp`

## What You Can Explore Freely

- How the current catalog works
- Firebase product/admin ideas
- Data model options
- Rough admin screen concepts
- How product customization, artwork, and verbiage should work
- Experimental commits on this branch

## Ask Andrew Before Doing These

- Merging anything into `main`
- Changing GitHub Pages deployment settings
- Replacing the Google Sheet workflow
- Creating or connecting production Firebase resources
- Turning on billing
- Adding payment/checkout features
- Deploying anything customer-facing
